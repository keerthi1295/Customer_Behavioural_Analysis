# Customer Behavior Data Analyst Portfolio Project

## Overview

This project focuses on analyzing customer purchasing behavior to uncover actionable business insights. Using Python, SQL, and Power BI, the analysis explores customer demographics, spending patterns, product preferences, review ratings, and customer loyalty segments. The goal is to help businesses understand their customers better and make data-driven decisions to improve retention and revenue.

## Objectives

* Analyze customer purchasing trends and shopping behavior.
* Identify high-value and loyal customers.
* Evaluate product performance based on customer reviews.
* Segment customers based on purchase history.
* Create interactive dashboards for business stakeholders.

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **MySQL** (Data querying and business analysis)
* **Power BI** (Interactive dashboards and visualizations)
* **Jupyter Notebook** (Data cleaning and exploratory analysis)

## Project Workflow

1. Data Collection and Understanding
2. Data Cleaning and Preprocessing using Python
3. Exploratory Data Analysis (EDA)
4. Business Analysis using SQL Queries
5. Customer Segmentation
6. Dashboard Development in Power BI
7. Insight Generation and Recommendations

## Key Business Questions Answered

* Which products have the highest customer ratings?
* What are the purchasing trends across customer segments?
* Who are the most loyal customers?
* Which product categories generate the highest sales?
* How do customer demographics influence purchasing behavior?
* What factors contribute to customer retention?

## Key Insights

* Identified top-performing products based on customer review ratings.
* Segmented customers into New, Returning, and Loyal categories.
* Analyzed spending patterns across different demographics.
* Highlighted customer groups with the highest purchase frequency.
* Discovered opportunities to improve customer retention and engagement.

## Dashboard Features

* Customer Demographics Overview
* Product Performance Analysis
* Customer Segmentation Dashboard
* Purchase Trend Analysis
* Customer Satisfaction Metrics
